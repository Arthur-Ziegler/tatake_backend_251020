## 认证方式

使用 **JWT Bearer Token** 认证：
```
Authorization: Bearer <token>
```

---

## 统一响应格式

```json
{
  "code": 200,        // 状态码
  "message": "success", // 响应消息
  "data": {}          // 响应数据（可选）
}
```

---

## 状态码说明

- `200`: 成功
- `400`: 请求参数错误
- `401`: 未认证
- `403`: 权限不足
- `404`: 资源不存在
- `409`: 资源冲突
- `422`: 请求参数验证失败
- `500`: 服务器内部错误
